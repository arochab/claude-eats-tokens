    Voir mes vrais chiffres

    Étape 1/5

    On va installer un petit outil sur ton ordinateur (pas sur le téléphone). C'est lui qui lit l'activité de Claude Code et l'envoie ici. À faire une seule fois.

    1 · Récupérer l'outil

    Sur ton ordinateur, télécharge le dossier du projet, puis dézippe-le quelque part où tu le retrouveras (ton Bureau, par exemple).

    Télécharger le dossier (.zip)

    Ce bouton ouvre GitHub sur ton ordinateur. Si tu lis ceci sur ton téléphone, ouvre plutôt cette page depuis ton PC — c'est là que l'outil doit vivre.

    2 · Créer ton compte

    Un compte relie ton ordinateur à cette page. Tu reçois un code de connexion (il commence par cet_). Pas de mot de passe.

    Compte créé. Voici ton code. Copie-le maintenant : tu vas le coller sur ton ordinateur à l'étape suivante.
    cet_…
    3 · Coller ton code sur l'ordinateur

    Dans le dossier que tu as dézippé, il y a un fichier secret.local.example.bat. Fais-en une copie et renomme-la en secret.local.bat.

    Ouvre secret.local.bat avec le Bloc-notes et colle ton code juste après le signe égal, comme ceci :

    set CET_API_KEY=cet_ton_code_ici

    Enregistre le fichier, puis ferme-le. C'est tout pour cette étape.

    4 · Lancer l'outil

    Sur Windows : double-clique sur DEMARRER.bat dans le dossier. Une fenêtre noire s'ouvre et se met à envoyer tes chiffres. Laisse-la ouverte.

    S'il te dit que Python manque : installe-le depuis python.org et, à l'écran d'installation, coche la case « Add Python to PATH ». Puis relance DEMARRER.bat.

    Sur Mac ou Linux : il n'y a pas de double-clic. Ouvre un terminal dans le dossier et lance python tools/push_usage.py --interval 60 (le fichier secret.local.bat te donne les valeurs à charger).

    5 · C'est branché

    Tant que la fenêtre reste ouverte sur ton ordinateur, tes chiffres arrivent tout seuls. Reviens ici : la démo laisse la place à tes vraies données, sans rien faire de plus.

    Rien ne s'affiche encore ? Laisse une minute à l'ordinateur pour le premier envoi, puis tire la page vers le bas pour rafraîchir. L'outil est à relancer seulement si tu redémarres l'ordinateur.

    5 € / mois. Résiliable quand tu veux.

    Arrête de rouvrir l'app pour savoir si tu peux relancer. Ton téléphone te prévient avant que ça bloque.

    • Sois prévenu à temps. Notifs dès 25, 50, 75 puis 90 % — pas seulement quand c'est trop tard.
    • Vois plus loin. 30 jours et tout l'historique, pas seulement les 7 derniers.
    • Anticipe la fin de mois. La projection te dit où tu atterris à ce rythme.
    • Fouille tes projets. Ouvre chaque projet : modèles, sessions, coût détaillé.
    • Waste Radar. Repère où part ton Opus (le modèle premium, donc le plus cher) quand un modèle plus léger aurait suffi — et ce que tu aurais pu économiser.
    • Boîte noire. Comprends enfin pourquoi ta fenêtre fond si vite — et quel projet est en cause.
    • Exporte tout. CSV et PNG pour garder tes chiffres ou les partager.
